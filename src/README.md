# AshkanTravel (Vue 3 + Vite)

یک نسخه‌ی بهبود یافته با:
- طراحی تمیز + دارک‌مود + RTL/LTR
- سیستم رنگ با تأکید بر اعتماد (آبی)، آرامش/طبیعت (سبز-آبی)، دعوت به اقدام (کهربایی)
- لیست تورها با **جستجو / فیلتر / مرتب‌سازی / صفحه‌بندی**
- صفحه جزئیات تور + برنامه سفر + فرم رزرو
- علاقه‌مندی‌ها (Persist در LocalStorage)
- Toast notifications
- ESLint + Prettier + Husky (اختیاری)

## اجرا

```bash
npm i
npm run dev
```

## کیفیت کدنویسی

```bash
npm run lint
npm run format
```

## فعال‌سازی Husky (اختیاری)

```bash
npx husky install
```
